# xd-es-cli

es脚手架工具  =>  兼容各个浏览器

<br>

## Getting Started

<br>

### Install

```shell
npm install xd-es-cli -g
```

<br>

###  Usage

```
Usage: xd-es-cli <command>

Options:
  -V, --version   output the version number
  -h, --help      display help for command

Commands:
  init|i          Generate a new project
  help [command]  display help for command
```
